## 第23课：安全口令验证器

> 对应课程：[第23课：循环里的停止与跳过口令](朵朵专属python课程.md#第23课循环里的停止与跳过口令)

```python
# 安全口令验证器
secret = "panda"

for chance in range(1, 4):
    password = input(f"第{chance}次尝试，请输入口令：")

    if password == "":
        print("没有输入内容，本次跳过。")
        continue

    if password == secret:
        print("口令正确，验证成功！")
        break

    print("口令错误，还剩", 3 - chance, "次机会。")
```

### 升级思考

三次都输错时，怎样增加一句“验证失败”？可以先在循环后根据一个布尔变量判断，或留到后续继续改造。

## 第24课：五次机会猜数字

> 对应课程：[第24课：猜数字大挑战](朵朵专属python课程.md#第24课猜数字大挑战)

```python
# 五次机会猜数字
import random

secret = random.randint(1, 20)
times = 0
is_correct = False

while times < 5:
    guess = int(input("猜一个1到20的数字："))
    times = times + 1

    if guess == secret:
        print("恭喜你猜对了！")
        print("你一共猜了", times, "次。")
        is_correct = True
        break
    elif guess < secret:
        print("太小了")
    else:
        print("太大了")

    print("还剩", 5 - times, "次机会。")

if not is_correct:
    print("本轮机会用完，正确答案是", secret)
```

## 第36课：石头剪刀布三局赛

> 对应课程：[第36课：让函数带着答案回来](朵朵专属python课程.md#第36课让函数带着答案回来)

```python
# 石头剪刀布三局赛
import random

choices = ["石头", "剪刀", "布"]

def is_valid_choice(choice):
    return choice in choices

def get_computer_choice():
    return random.choice(choices)

def decide_winner(player, computer):
    if player == computer:
        return "平局"
    elif (
        (player == "石头" and computer == "剪刀")
        or (player == "剪刀" and computer == "布")
        or (player == "布" and computer == "石头")
    ):
        return "玩家"
    else:
        return "电脑"

player_score = 0
computer_score = 0

print("欢迎参加石头剪刀布三局赛！")

for round_number in range(1, 4):
    print(f"\n第{round_number}局")
    player = input("请输入石头、剪刀或布：")

    if not is_valid_choice(player):
        print("输入无效，本局不得分。")
        continue

    computer = get_computer_choice()
    result = decide_winner(player, computer)

    print("电脑出了：", computer)

    if result == "玩家":
        player_score = player_score + 1
        print("这一局你赢了！")
    elif result == "电脑":
        computer_score = computer_score + 1
        print("这一局电脑赢了！")
    else:
        print("这一局是平局！")

    print("当前比分：玩家", player_score, "：电脑", computer_score)

print("\n比赛结束")
print("最终比分：玩家", player_score, "：电脑", computer_score)

if player_score > computer_score:
    print("恭喜你成为冠军！")
elif computer_score > player_score:
    print("电脑获得冠军，再挑战一次吧！")
else:
    print("双方打成平局！")
```
