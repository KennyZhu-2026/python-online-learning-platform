# 朵朵专属 Python 课程大纲

> 适用对象：小学三年级、零编程基础  
> 学习方式：孩子自学为主，家长启发和指导为辅  
> 课程形式：以知识单元划分，不规定每课时长；达到过关标准后再进入下一单元

## 一、课程目标

完成核心课程后，朵朵能够：

- 读懂并编写简单的 Python 程序。
- 使用 `print()` 输出信息，使用 `input()` 获取用户输入。
- 使用变量保存文字、数字和判断结果。
- 区分字符串、整数、浮点数和布尔值等基本数据类型。
- 使用数学运算、比较运算和逻辑运算解决简单问题。
- 使用 `if`、`if/else`、`if/elif/else` 处理不同情况。
- 使用 `for`、`range()` 和 `while` 解决重复任务。
- 使用列表和字典管理多项数据。
- 使用函数组织代码，并理解参数和返回值。
- 根据常见报错提示检查简单错误。
- 在参考旧代码的情况下，独立完成一个简单小游戏。

## 二、课程设计原则

1. **一课一重点**：每个学习单元只突出一个核心知识点。
2. **不限定时长**：短知识点可以一次完成，较难内容可以分多次学习。
3. **先猜再运行**：运行示例前，先让朵朵预测程序结果。
4. **先模仿再创造**：先运行示例，再修改示例，最后完成自己的作品。
5. **允许查语法卡**：不要求机械背诵或从空白页面默写完整代码。
6. **作品驱动**：每个阶段完成一个可以使用、讲解或展示的小项目。
7. **错误也是课程**：将调试称为“程序侦探”，培养独立发现问题的习惯。

## 第四阶段：比较与条件判断

### 第12课：比较运算

**核心知识点**

- 等于 `==`。
- 不等于 `!=`。

**核心知识点**

- 条件成立时执行 `if`。
- 条件不成立时执行 `else`。
- 两个分支中只会执行一个。

**最小示例**

```python
password = input("请输入密码：")
```

- 程序按从上到下的顺序判断。
- 执行第一个成立的分支。
- 使用 `else` 处理其余情况。

**最小示例**

```python
score = int(input("请输入分数："))
```

**最小示例**

## 第六阶段：`while` 循环

### 第22课：认识 `while` 循环

**核心知识点**

- 条件成立时不断重复。
- 设置循环的初始状态。
- 在循环中更新变量。
- 识别和避免无限循环。

**最小示例**

```python

```

**过关任务**

### 第23课：循环控制

**核心知识点**

- `break`：立即结束整个循环。
- `continue`：跳过本次循环，继续下一次。
- 判断循环应该在什么时候停止。

**最小示例**

```python
for number in range(1, 11):
    if number == 6:
        break
    print(number)
```

```python
for number in range(1, 6):
    if number == 3:
        continue
    print(number)
```

**过关任务**

让程序跳过一个指定数字，并在遇到另一个指定数字时停止。

### 第24课：`while` 综合应用

**核心知识点**

- 让用户反复输入。
- 使用条件决定是否继续。
- 配合 `if` 和 `break` 控制游戏。

**阶段项目：猜数字游戏**

```python
import random

secret = random.randint(1, 10)

while True:
    guess = int(input("猜一个1到10的数字："))

    if guess == secret:
        print("猜对了！")
        break
    elif guess < secret:
        print("太小了")
    else:
        print("太大了")
```

进阶改造：增加猜测次数，并在猜对后显示总共猜了几次。

## 第九阶段：函数

### 第34课：函数基础

**核心知识点**

- 函数是一组可以重复使用的代码。
- 使用 `def` 定义函数。
- 调用函数。
- 函数体需要缩进。
- 定义函数时，函数不会自动执行。

**最小示例**

```python
def say_hello():
    print("你好！")
    print("欢迎学习 Python！")

say_hello()
say_hello()
```

**过关任务**

创建三个“机器人技能”函数，并分别调用它们。

### 第35课：函数参数

**核心知识点**

- 参数是交给函数的数据。
- 使用一个参数。
- 使用多个参数。
- 参数让函数能够处理不同数据。

**最小示例**

```python
def greet(name):
    print("你好，", name)

def add(a, b):
    print(a + b)

greet("朵朵")
add(3, 5)
```

**过关任务**

制作“个性化问候机器人”，让函数可以向不同的人问好。

### 第36课：函数返回值

**核心知识点**

- 使用 `return` 返回结果。
- 将函数返回值保存到变量。
- 区分“打印结果”和“返回结果”。
- 函数运行到 `return` 时会结束。

**最小示例**

```python
def add(a, b):
    return a + b

answer = add(3, 5)
print(answer)
```

### 基础版参考结构

```python
import random

choices = ["石头", "剪刀", "布"]

def get_computer_choice():
    return random.choice(choices)

def decide_winner(player, computer):
    if player == computer:
        return "平局"
    elif (
        (player == "石头" and computer == "剪刀")
        or (player == "剪刀" and computer == "布")
        or (player == "布" and computer == "石头")
    ):
```

## 七、每个自学单元的统一模板

为了让朵朵可以逐步形成自学习惯，每个单元建议采用相同结构。

### 1. 本课目标

只设置一个可以检查的主要目标，例如：

> 学完这一课，我能用变量保存和修改一个人的分数。

### 2. 生活类比

用孩子熟悉的事物解释概念，例如：

- 变量：贴了名字的盒子。
- 列表：有编号的抽屉或排队名单。
- 字典：姓名和电话号码组成的通讯录。
- 条件判断：根据天气决定带雨伞还是太阳帽。
- 循环：连续拍手五次。
- 函数：给一组动作起一个“技能名称”。

### 3. 最小示例

新知识的第一个示例尽量控制在 3～8 行。

### 4. 预测与观察

先让朵朵猜测程序结果，再运行验证。建议问题：

- 这段程序会显示什么？
- 哪一行会先执行？
- 修改这个数字后会发生什么？
- 删除这一行后程序还能运行吗？

### 5. 模仿练习

照着示例重新完成一次，可以查看示例，不要求默写。

### 6. 修改挑战

每课至少修改三处，例如：

- 修改文字。
- 修改数字。
- 修改变量名。
- 增加一个新条件。
- 增加一个列表元素。
- 调整循环次数。

### 7. 错误侦探

每课准备一段包含 1～2 个错误的代码，重点检查：

- 是否使用了中文标点。
- 引号和括号是否成对。
- 是否忘记冒号。
- 缩进是否正确。
- 变量名拼写和大小写是否一致。
- 数字输入是否进行了类型转换。

### 8. 过关作品

完成一个可以运行、修改和展示的小程序。

### 9. 自我检查

- [ ] 我能运行示例。
- [ ] 我能大致说出每一行的作用。
- [ ] 我能修改示例并得到不同结果。
- [ ] 我能完成本课的过关任务。
- [ ] 我能向家长讲解程序是怎样工作的。

## 八、学习效果分级

| 等级 | 表现 |

## 九、家长指导建议

家长的重点不是直接给出答案，而是帮助朵朵观察和推理。可以经常问：

- 这个变量里面保存了什么？
- `input()` 得到的是文字还是数字？
- 这个条件什么时候成立？
- 两个分支中会执行哪一个？
- 这段循环会重复几次？
- 哪一行改变了变量的值？
- 列表中第一个元素为什么是编号 `0`？
- 函数收到了什么参数？
- 函数最后返回了什么？
- 报错信息指出的是哪一行？

遇到错误时，推荐按照以下顺序检查：

1. 查看报错发生在哪一行。
2. 检查英文引号、括号和冒号。
3. 检查缩进。
4. 检查变量名拼写和大小写。
5. 检查数据类型是否合适。
6. 一次只修改一个地方，再重新运行。
