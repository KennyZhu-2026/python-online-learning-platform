# 第1课：启动 Python 魔法盒

## 学习目标

认识 Python 程序的基本样子，能够创建、保存并运行第一段代码。学会用注释给自己留下说明。

**本课涉及的知识点**

- 运行 `.py` 文件
- 代码执行顺序
- 注释 `#`
- 英文标点

## 知识点讲解

程序就像写给电脑的一张“任务清单”。我们把每个任务写成一行代码，电脑通常会从第一行开始，一行一行向下执行。

Python 文件的名字通常以 `.py` 结尾，比如 `hello.py`。写完代码后，需要点击“运行”，电脑才会按照清单开始工作。

代码中以 `#` 开头的内容叫作**注释**。注释像写在课本边上的小提醒，是给人看的，电脑运行时会跳过它。

需要特别注意：Python 代码通常要使用英文引号、英文括号和英文冒号。中文标点虽然看起来很像，但电脑会把它们当成不同的符号。

## 代码示例

运行前先猜一猜：电脑会显示几行文字？注释会显示吗？

```python
# 我的第一个 Python 程序
print("朵朵开始学习 Python 啦！")
print("今天是第1课。")
```

**运行结果示例**

```text
朵朵开始学习 Python 啦！
今天是第1课。
```

## 实操练习

复制代码示例，然后完成下面三项修改：

1. 把“朵朵”改成自己的昵称。
2. 把“第1课”改成“我的第一段程序”。
3. 在第一行增加一条注释，说明这个程序是谁写的。

参考结构：

```python

print("这里写第一句话")
print("这里写第二句话")
```

# 第2课：让电脑开口说话

## 学习目标

学会让电脑输出文字、数字和计算结果，并发现引号内外内容的不同。

**本课涉及的知识点**

- `print()`
- 输出字符串
- 输出数字
- 输出多个内容

## 知识点讲解

`print()` 是 Python 的“扩音器”。我们把想让电脑显示的内容放进括号里，电脑就会把它输出到屏幕上。

文字需要放在引号里，因为引号相当于给文字套上一层保护罩，告诉电脑：“这里面的内容请原样显示。”数字如果不加引号，电脑会把它当成可以计算的数。

一次 `print()` 可以输出多个内容，中间用英文逗号隔开。Python 会在这些内容之间自动留出空格。

## 代码示例

```python
print("朵朵")
print(3 + 5)
print("我今年", 9, "岁")
```

**运行结果示例**

```text
朵朵
8
我今年 9 岁
```

重点比较下面两行：

```python
print("3 + 5")
print(3 + 5)
```

**运行结果示例**

```text
3 + 5
8
```

第一行把 `3 + 5` 当成文字，第二行会真的计算加法。

## 实操练习

运行下面的代码，再按照要求修改：

```python

```

修改任务：

2. 把加法题换成另一道题。
3. 在星期三后面增加星期四。
4. 增加一行，输出自己的学校或班级。

# 第3课：给输出穿上整齐外衣

## 学习目标

学会给输出内容换行、留出间隔并进行简单格式化，让程序显示得更整齐。

**本课涉及的知识点**

- 换行符 `\n`
- 制表符 `\t`
- f-string 初体验

## 知识点讲解

写贺卡和课程表时，我们会用换行、空格和对齐让内容更整齐。程序输出也可以排版。

`\n` 像“回车键”，会让后面的内容换到下一行；`\t` 像按了一次较宽的 Tab 键，可以在内容之间留出较大的空隙。

有时我们还会在字符串前面加一个字母 `f`，写成 f-string。它像一张有空格的填空卡，大括号 `{}` 中可以放入要显示的内容。这一课先认识它，第4课学完变量后会再次使用。

## 代码示例

```python
print("姓名：朵朵\n年龄：9岁")
print("语文\t数学\t英语")
print(f"三盒彩笔一共有{3 * 12}支")
```

**运行结果示例**

```text
姓名：朵朵
年龄：9岁
语文    数学    英语
三盒彩笔一共有36支
```

## 实操练习

复制并修改下面的代码：

```python

print("星期一\t星期二\t星期三")
print(f"5个小组，每组4人，一共有{5 * 4}人")
```

修改任务：

1. 把三餐内容换成自己喜欢的食物。
2. 在星期三后面增加星期四。
3. 把人数题改成“6个小组，每组5人”。

# 第4课：会变身的数据盒子

## 学习目标

理解变量是会保存数据的“标签盒子”，能够创建、读取和修改变量。

**本课涉及的知识点**

- 变量
- 赋值符号 `=`
- 变量更新
- 变量命名

## 知识点讲解

变量就像一个贴着标签的小盒子。标签是变量名，盒子里面可以放文字、数字或其他数据。

例如 `score = 10` 表示找到一个名叫 `score` 的盒子，把数字 `10` 放进去。等号 `=` 在这里不是“左右相等”，而是“把右边的东西放进左边的盒子”。

变量里的内容可以改变。如果后来执行 `score = 20`，就像把盒子里的 `10` 拿走，换成了 `20`。

变量名要有意义，最好使用英文单词或简单拼音。变量名不能以数字开头，中间不能有空格，并且 `score` 和 `Score` 会被当成两个不同的名字。

## 代码示例

```python
score = 10
print(score)

score = 20
print(score)
```

**运行结果示例**

```text
10
20
```

输出结果是 `10` 和 `20`，因为第二次赋值更换了盒子里的内容。

## 实操练习

```python
name = "朵朵"
age = 9
hobby = "画画"

print(name)
print(age)
print(hobby)
```

修改任务：

2. 增加一个保存喜欢动物的变量 `animal`。
3. 把 `age` 加 1 后再输出。

提示：

```python

```

# 第5课：数据王国的四位居民

## 学习目标

认识 Python 中四种常见的数据类型，并能用 `type()` 检查数据属于哪一类。

**本课涉及的知识点**

- 字符串 `str`
- 整数 `int`
- 浮点数 `float`
- 布尔值 `bool`
- `type()`

## 知识点讲解

整理物品时，我们会把书放进书柜、衣服放进衣柜、玩具放进玩具箱。电脑也会给数据分类，这些类别叫作“数据类型”。

初学 Python 时，最常见的四种类型是：

- `str`：字符串，也就是文字，如 `"朵朵"`。
- `int`：整数，如 `9`、`100`、`-3`。
- `float`：小数，如 `135.5`、`3.14`。
- `bool`：布尔值，只有 `True` 和 `False`，表示“是/否”“成立/不成立”。

`type()` 像一个数据检查器，可以告诉我们某个数据属于哪一类。

注意：`"9"` 有引号，是字符串；`9` 没有引号，是整数。它们看起来相似，但电脑会用不同的方式处理。

## 代码示例

```python
name = "朵朵"
age = 9
height = 135.5
is_student = True

print(type(name))
print(type(age))
print(type(height))
print(type(is_student))
```

**运行结果示例**

```text
<class 'str'>
<class 'int'>
<class 'float'>
<class 'bool'>
```

## 实操练习

给下面四个变量选择合适的数据，并运行 `type()` 检查：

```python
favorite_food = "草莓"
book_count = 12
temperature = 26.5
has_homework = False

print(type(favorite_food))
print(type(book_count))
print(type(temperature))
print(type(has_homework))
```

修改任务：

2. 修改书本数量和温度。
3. 如果今天有作业，把 `False` 改成 `True`。
4. 比较 `type("12")` 和 `type(12)` 的结果。

# 第6课：让电脑听见你的回答

## 学习目标

学会使用 `input()` 向用户提问，并把回答保存到变量中供后面的代码使用。

**本课涉及的知识点**

- `input()`
- 输入提示语
- 保存用户输入
- 输入结果是字符串

## 知识点讲解

以前的程序只会按照我们提前写好的内容说话。从这一课开始，程序可以向用户提问，并根据用户的回答继续工作。

`input()` 像一个“等待回答的话筒”。程序运行到这里时会停下来，等用户输入内容并按下回车键。

输入的答案通常要保存到变量里，否则程序听到了回答，却没有给回答贴上标签，后面就不方便使用。

无论用户输入的是姓名还是数字，`input()` 默认都会把它当成字符串。数字转换会在下一课学习。

## 代码示例

```python
name = input("你叫什么名字？")
print("你好，", name)
```

**运行结果示例**

```text
你叫什么名字？朵朵
你好， 朵朵
```

## 实操练习

```python
animal = input("你最喜欢什么动物？")
food = input("它喜欢吃什么？")

print("你喜欢的动物是", animal)

```

# 第7课：给数据换件合适的衣服

## 学习目标

理解同一份数据有时需要转换类型，能够在字符串、整数和小数之间进行基础转换。

**本课涉及的知识点**

- `int()`
- `float()`
- `str()`
- 输入与类型转换

## 知识点讲解

同样是一张写着“9”的卡片，它可以表示数字九，也可以只是一段文字。`input()` 收到的内容默认都是文字，所以直接用输入的年龄做加法，电脑可能不知道我们真正想算什么。

类型转换就像给数据换一套合适的衣服：

- `int()` 把合适的内容转换成整数。
- `float()` 把合适的内容转换成小数。
- `str()` 把内容转换成字符串。

只有看起来像数字的文字才能转换成数字。比如 `"12"` 可以变成整数，`"苹果"` 不能变成整数。

## 代码示例

```python
age_text = input("你今年几岁？")
age = int(age_text)

print("明年你就", age + 1, "岁了")
```

**运行结果示例**

```text
你今年几岁？9
明年你就 10 岁了
```

也可以把输入和转换写在同一行：

```python
age = int(input("你今年几岁？"))
```

**运行结果示例**

```text
你今年几岁？9
```

## 实操练习

```python
number = int(input("请输入一个整数："))
price = float(input("请输入商品价格："))

print("这个整数加1是", number + 1)
print("两件商品一共", price * 2, "元")
```

修改任务：

1. 把“加1”改成“加10”。
2. 把“两件商品”改成“三件商品”。
3. 使用 `str()` 把数字转换成字符串，再用 `type()` 检查。

# 第8课：口袋里的超级计算器

## 学习目标

学会把 Python 当作计算器，使用常见运算符解决分糖果、购物等数学问题。

**本课涉及的知识点**

- `+ - * /`
- 整除 `//`
- 取余 `%`
- 乘方 `**`
- 运算顺序

## 知识点讲解

Python 不只是会说话的机器人，也是一台很快的计算器。常见运算符就像不同的计算按钮：

- `+` 加法。
- `-` 减法。
- `*` 乘法。
- `/` 普通除法，结果通常会带小数。
- `//` 整除，只保留完整的份数。
- `%` 取余数，看看平均分完后还剩多少。
- `**` 乘方，例如 `2 ** 3` 表示三个 2 相乘。

可以把 `//` 和 `%` 想象成分糖果：17颗糖平均分给5个人，每人得到 `17 // 5` 颗，还剩 `17 % 5` 颗。

括号和数学课中的作用相似，括号里的内容会先计算。

## 代码示例

```python
candies = 17
children = 5

print("每人分到", candies // children, "颗")
print("还剩", candies % children, "颗")
```

**运行结果示例**

```text
每人分到 3 颗
还剩 2 颗
```

## 实操练习

```python
a = int(input("请输入第一个数字："))
b = int(input("请输入第二个数字："))

print("加法：", a + b)
print("减法：", a - b)
print("乘法：", a * b)
```

修改任务：

1. 增加普通除法的结果。
2. 增加整除和余数的结果。
3. 增加 `a ** 2`，显示第一个数字的平方。
4. 输入不同的数字，观察结果。

# 第9课：文字小火车出发啦

## 学习目标

认识字符串的拼接、重复和长度，让多段文字像小火车一样组合起来。

**本课涉及的知识点**

- 字符串
- 字符串拼接 `+`
- 字符串重复 `*`
- `len()`

## 知识点讲解

字符串就是一串文字，可以是一个字、一句话，也可以是看起来像数字的内容。它们通常住在单引号或双引号里面。

字符串也能做一些有趣的“运算”：

- 使用 `+` 可以把两段文字接起来，像拼接两节小火车。
- 使用 `*` 可以让同一段文字重复多次。
- 使用 `len()` 可以数出字符串里有多少个字符。

拼接时，两边都必须是字符串。字符串和整数不能直接用 `+` 相连，需要先转换类型或使用 f-string。

## 代码示例

```python
first_name = "朵"
last_name = "朵"
name = first_name + last_name

print(name)
print("哈" * 3)
print(len(name))
```

**运行结果示例**

```text
朵朵
哈哈哈
2
```

## 实操练习

```python
animal = "熊猫"
food = "竹子"
sentence = animal + "喜欢吃" + food

print(sentence)
print("加油！" * 3)
print("句子长度是", len(sentence))
```

修改任务：

```text
检查结束
```

```text
优秀
```

# 第19课：启动数字排队机

## 知识点讲解

`range()` 像一台“数字排队机”，可以按照规则生成一串数字，常常和 `for` 一起使用。

## 代码示例

```python
for number in range(1, 6):
    print(number)
```

```text
1
2
3
4
5
```

## 实操练习

```python
print("从1数到10：")

    print(number)
```

# 第22课：只要条件成立就继续

## 学习目标

认识 `while` 循环，理解“只要条件成立就继续”的运行方式，并避免无限循环。

**本课涉及的知识点**

- `while`
- 循环条件
- 更新变量
- 无限循环

## 知识点讲解

`for` 适合已经知道要重复多少次或要依次处理什么内容的任务。`while` 表示“只要条件还成立，就继续重复”，适合不知道确切次数的情况。

例如“只要水桶还没有装满，就继续倒水”。我们不一定提前知道要倒几次，但知道什么时候应该停止。

一个常见的 `while` 循环包含三部分：

1. 循环开始前设置变量。
2. 用条件决定是否继续。
3. 在循环中更新变量，让条件最终能够变成 `False`。

如果忘记更新变量，程序可能永远停不下来，这叫作无限循环。

## 代码示例

```python
count = 1

while count <= 5:
    print(count)
    count = count + 1
```

**运行结果示例**

```text
1
2
3
4
5
```

## 实操练习

```python

    print(number)
    number = number - 1

print("发射！")
```

修改任务：

```python
    print(number)
```

```text
2
3
4
5
```

```python
for number in range(1, 6):
    if number == 3:
        continue
    print(number)
```

**运行结果示例**

```text
1
2
4
5
```

## 实操练习


# 第29课：依次点名列表成员

```text
欢迎参观 兔子 馆

```

## 实操练习


# 第30课：切片、排序与排行榜

## 学习目标

学会切取列表的一部分，并使用排序和统计工具整理一组数字。

**本课涉及的知识点**

- 列表切片
- `sort()` 与 `sorted()`
- `reverse()`
- `min()`、`max()`、`sum()`

## 知识点讲解

列表切片像从一排积木中取出连续的一段。它的写法和字符串切片相似：`numbers[1:4]` 从索引1开始，取到索引4之前。

常见的列表工具包括：

- `sort()`：直接把原列表排序。
- `reverse()`：直接把原列表前后翻转。
- `sorted()`：生成一个排序后的新列表，原列表不变。
- `min()`：找最小值。
- `max()`：找最大值。
- `sum()`：求数字列表的总和。

`numbers[:]` 可以复制出一个新列表。它们内容相同，但不是同一个列表盒子。

## 代码示例

```python
numbers = [9, 3, 7, 1]

print(numbers[1:3])
print(sorted(numbers))
print(numbers)
```

**运行结果示例**

```text
[3, 7]
[1, 3, 7, 9]
[9, 3, 7, 1]
```

## 实操练习

```python
scores = [88, 95, 72, 100, 84]

print("前三项：", scores[0:3])
print("最高分：", max(scores))
print("最低分：", min(scores))
print("总分：", sum(scores))
print("从低到高：", sorted(scores))
```

修改任务：

2. 使用 `sort()` 后再次输出原列表。

# 第31课：会查资料的魔法字典

## 学习目标

认识用“键—值”保存资料的字典，并能通过键准确找到对应内容。

**本课涉及的知识点**

- 创建字典
- 键 `key`
- 值 `value`
- 通过键访问数据

## 知识点讲解

列表通过位置编号寻找数据，字典则通过“名称”寻找数据。它很像一本真正的字典：看到一个词，就能找到对应的解释。

字典中的每一组数据叫作“键值对”：

- 键 `key` 像标签，例如 `"name"`。
- 值 `value` 是标签对应的内容，例如 `"朵朵"`。

字典使用英文大括号 `{}`，键和值之间用英文冒号连接，不同键值对之间用英文逗号隔开。

读取时写 `student["name"]`，相当于告诉电脑：“请在学生档案里找到 name 标签对应的内容。”键的拼写必须正确，否则会出现错误。

## 代码示例

```python
student = {

    "age": 9,
    "hobby": "画画"
}

print(student["name"])
print(student["age"])
```

**运行结果示例**

```text
朵朵
9
```

## 实操练习

```python
pet = {
    "name": "团团",
    "animal": "兔子",

    "food": "胡萝卜"
}

print("宠物名字：", pet["name"])

print("喜欢的食物：", pet["food"])
```

# 第33课：让字典逐项自我介绍

## 学习目标

学会遍历字典的键、值或键值对，把整份资料逐项展示出来。

**本课涉及的知识点**

- `keys()`
- `values()`
- `items()`
- 遍历字典

## 知识点讲解

字典中可能有很多键值对。如果一个个写键来读取会很麻烦，所以可以使用循环遍历。

字典提供三个常用工具：

- `keys()`：取得所有键。
- `values()`：取得所有值。
- `items()`：同时取得每一组键和值。

遍历 `items()` 时通常使用两个循环变量：第一个接收键，第二个接收值，就像每次从档案盒里拿出一张“标签＋内容”的卡片。

## 代码示例

```python
student = {
    "name": "朵朵",
    "age": 9,
    "hobby": "画画"
}

for key, value in student.items():
    print(key, "：", value)
```

**运行结果示例**

```text
name ： 朵朵
age ： 9
hobby ： 画画
```

## 实操练习

```python
scores = {
    "语文": 95,
    "数学": 98,
    "英语": 92
}

print("所有科目：")
for subject in scores.keys():
    print(subject)

print("所有成绩：")
for score in scores.values():
    print(score)
```

# 第35课：给函数递一张任务卡

- 重复使用函数

## 知识点讲解

参数像快递盒上的收件人空格。定义函数时先留下位置，调用函数时再填入实际内容。

```python
def greet(name):
```

```python
greet("朵朵")
greet("妈妈")
```

```text
你好， 朵朵
你好， 妈妈
```

```python
def show_pet(name, animal):
```

# 第36课：让函数带着答案回来

## 学习目标

理解函数怎样用 `return` 把结果交回来，并能把返回值用于后续判断和计算。

**本课涉及的知识点**

- `return`
- 函数返回值
- 保存返回结果
- `print()` 与 `return` 的区别

## 知识点讲解

有些函数只负责表演动作，有些函数则像一台加工机器：把原料放进去，经过处理后，把结果送出来。送出结果要使用 `return`。

例如把两个数字交给加法函数，函数计算后用 `return` 把答案交回来。程序可以把返回值保存到变量，也可以继续用于判断或计算。

`print()` 是把内容显示给人看，`return` 是把结果交还给调用函数的程序。它们的用途不同。

函数运行到 `return` 时会立即结束，写在同一条执行路线的 `return` 后面的代码不会再执行。

## 代码示例

```python
def add(a, b):
    return a + b

answer = add(3, 5)
print(answer)
```

**运行结果示例**

```text
8
```

## 实操练习

```python
def double(number):
    return number * 2

def is_even(number):
    return number % 2 == 0

value = int(input("请输入一个整数："))

print("两倍是：", double(value))
print("是偶数吗：", is_even(value))
```

修改任务：

1. 增加返回平方的 `square()` 函数。
2. 增加返回两个数字中较大值的函数。
3. 把一个函数的返回值保存到变量，再继续计算。
4. 把 `return` 临时改成 `print()`，观察函数外部得到的结果有什么变化。
